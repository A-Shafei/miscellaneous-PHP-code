Web Release vTEST

Released on 13th of March, here's what this release has changed:

  1. A major new feature is here! Called the Unlimited Listings Package; users will, soon, be able to buy a new package that lasts a limited time; that, while active, would allow them to add a limited number of listings everyday without deducting from their credit!
  2. Changed the way we sort the trending questions in the Questions page (that's https://egypt.aqarmap.com/en/question/). We now value how new the question is, in addition to how many answers it has.
  3. Fixed an issue with users getting an email that their listings have been approved, while the listings are still pending photos and haven't been reviewed by an admin.
